## v0.6.0 - 2026-06-22
### Changed
* Bumped dependencies to their latest versions, including terraform-plugin-sdk/v2 and the migration of blang/semver to the v4 module path
### Fixed
* Fixed a potential crash when parsing Dokku output lines that contain no colon (e.g. config:show, domains:report)
* Fixed a slice-allocation bug that prepended empty strings to domains, buildpacks and ports, which could leak empty values into Dokku commands
* Surface the error returned when re-reading an app immediately after creation instead of risking a nil dereference
### Security
* Validate and shell-escape app names, domains and port mappings before they are interpolated into SSH commands, preventing command injection via crafted resource attributes
